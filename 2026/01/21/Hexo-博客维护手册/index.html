Hexo 博客维护手册

1. 环境检查

如果换了电脑或重装系统，请先执行：

  • 安装依赖：在项目根目录下运行 npm install。
  • 全局工具：确保已安装 npm install -g hexo-cli。

2. 核心命令手册

功能 命令 简写 备注
新建文章 hexo new "标题" hexo n 在 source/_posts 生成 .md 文件
清理缓存 hexo clean 无 遇到页面不更新时必做
生成静态页 hexo generate hexo g 将 Markdown 转为 HTML
启动预览 hexo server hexo s 默认地址：http://localhost:4000
部署发布 hexo deploy hexo d 上传至 GitHub Pages

4. 常见问题备忘

  • 多标签写法：tags: [标签1, 标签2]。
  • 图片存放：建议放在 source/images/，文中引用：![描述](/images/xxx.png)。
  • 配置文件：
  • _config.yml (根目录)：改网站名、改 GitHub 地址。
  • _config.yml (主题目录)：改菜单、改头像、改评论插件。
